# Local rewrite engine

The alpha engine performs three distinct layers:

1. **Context-preserving rewrite**
   - Rephrases accusatory or vague statements sentence by sentence.
   - Keeps names, project references, dates and URLs present in the original.
   - Converts pressure into useful questions where possible.

2. **Communication adaptation**
   - Cool Blue: facts, open actions, ownership, timing and documentation.
   - Fiery Red: requested action, decision and deadline first.
   - Sunshine Yellow: short personal opener, positive framing and invitation to contribute.
   - Earth Green: alignment, support, involved colleagues and team impact.
   - Mixed profiles combine the dominant and secondary energies.

3. **Separate advice**
   - Recommends adding actual documents or URLs but never invents them.
   - Recommends a call or meeting when writing is likely to create more friction.

A production version should replace this deterministic engine with CARYA's internally trained agent while retaining the same UI contract.
