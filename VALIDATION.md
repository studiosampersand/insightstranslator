# Validation

Validated locally:

- JavaScript syntax for the PWA, Teach Liz page and Node workflow scripts.
- JSON parsing for scenarios, conversation index, seed conversation and review index.
- YAML parsing for all workflow and issue-form files.
- Exactly 50 fictional personas; every profile totals 100% and has one dominant energy above 50%.
- Seed conversation loading references.
- Review issue parser creates a structured review and candidate approved example.

Not executed locally:

- A live GitHub Models inference request.
- GitHub Actions permission changes.
- Automatic pull-request creation in the user's repository.
- GitHub Pages deployment in the user's repository.

Those require the repository's GitHub environment and permissions.
