# Prompt tuning rules

- Keep organization-wide professional rules separate from personal preferences.
- Do not treat one person's preferences as the rule for everyone with the same dominant energy.
- Preserve facts and intended action before adapting style.
- Use dominant and secondary energies as communication guidance, not stereotypes.
- When sender and recipient differ strongly, prefer common ground.
- Advice such as adding documentation or planning a meeting stays outside the rewritten message unless the source already contains that fact.
- Never invent URLs, support, approvals, deadlines or organizational consensus.
- Evaluate prompt changes against a fixed set of reviewed examples before merging.
