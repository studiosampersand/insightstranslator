# Contact actions in the public demo

The public GitHub Pages version deliberately does not embed a real employee email address or Microsoft 365 user principal name.

Therefore:

- “Open Teams” opens Microsoft Teams, but does not start a chat with a named employee.
- “Compose email” opens a prefilled email draft without a recipient.
- Maarten is shown as the HR and fleet contact, as requested for the concept demonstration.
- The production Microsoft version should populate the exact Teams deep link and email recipient from the approved internal directory.
- “Open HaloPSA” directly opens `https://horizon.carya.tools/auth`.
